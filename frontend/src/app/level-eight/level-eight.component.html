To create three records simultaneously in a single SQL query, you can use the INSERT INTO statement with a VALUES block containing the values for the new records.

Here is an example:

INSERT INTO table_name ( 1._attribute_name , 2._attribute_name , 3._attribute_name )
VALUES
( 1.entry_valueOf_1._attribute , 1.entry_valueOf_2._attribute , 1.entry_valueOf_3._attribute ),
( 2.entry_valueOf_1._attribute , 2.entry_valueOf_2._attribute , 2.entry_valueOf_3._attribute ),
( 3.entry_valueOf_1._attribute , 3.entry_valueOf_2._attribute , 3.entry_valueOf_3._attribute );
